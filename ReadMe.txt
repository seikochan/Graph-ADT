2. Use a command prompt/terminal, navigate to the directory where you unzipped the files to.
3. Run the command: "javac *.java" to compile all the java files into executable .class files.
4. Run command: "java DirectedGraphDriver ./testFiles/vnaFileName.vna". You may change the file name to any of those above listed under "Test Files"
